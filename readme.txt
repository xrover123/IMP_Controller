Программа контроля импорта файлов. Имеет свой шедулер позволяющий запускаться и останавливаться в определенное время.
Из "АйТи какой Аптеки" файлы передаются группами (чек, спецификация, платежи). В одной группе файлов содержится несколько чеков за прошедший период.
В рабочее время с определенным интервалом программа проверяет по маске файлы в заданных дирректориях.
После нахождения первого файла из списка (и первого по времени изменения файла) выделяет ID файловой группы (в масках задается символом "#").
В маски остальных файлов добавляется ID группы вместо символа "#" и дальше поиск вгруппе происходит по этим маскам. 
После нахождения всех файлов в группе программа начинает последовательно (по списку) анализировать изменямость файла.
После того как файл прекратил изменятся программа перемещает файл во временную дирректорию и оттуда перещает файл в дирректорию бекапа.
Далее создается список перемещенных во временную дирректорию файлов группы и этот список записывается в файл на диск.
После чего запускается программа импорта файлов группы ей передается в качестве параметров: 
  1-строка подключения к БД;
  2-имя файла со списком файлов из группы. (строка списка файорв имеет вид: F<№ПП>=<полное название файла>);
Настройка ведется через конфигурационный файл exchange.ini (классический формат ini-файла):

[DB]
DB_NAME                      - База данных (псевдоним).
USER                         - Пользователь БД
PSW                          - Пароль в зашифрованном виде (шифруется программой psw.exe <пароль> из каталога запуска самой программы).
[OTHERS]
IMP_LOG=c:\imp.log           - Основной лог-файл.
IMP_ANSW=c:\ansver.txt       - Текст ответа от программы импорта группы файлов (полное имя файла). Выдержки или весь файл переписыватся в основной LOG-файл.
IMP_PROG=c:\IMP.EXE          - Программа импорта (полное имя файла).
[SHEDULER]
IMP_DEY                      - Рабочие дни недели (начиная с понедельника): 1,2,3,4,5,6
SH_BGN	                     - Начало рабочего дня
SH_END	                     - Окончание рабочего дня
SH_INT	                     - Интервал межу итерациями поиска групп файлов (с).
[FILE]
IMP_FILE_MOVE_ATT            - Кол-во попыток перемещения файла (после определения неизменяемости файла).
IMP_FILE_MOVE_SLEEP          - Период между попытками перемещения файла (с).
IMP_FILE_MOVE_TIME           - Период ожидания следующих за первым файлов (с).
IMP_FILE_WAIT_MOVE           - Ожидание стабильности (неизменяемости) файла (с)
IMP_FILE_CHECK_INTERVAL      - Интервал проверки стабильности файла (мс).
IMP_FILE_MOVE_TMP            - Временная дирректория (в которую будут перемещены файлы группы) для обработки программой импорта файлов.

Примечание:
  Комментарии в ini-файле не предусмотрены.
  Строка подключения для программы импорта формируется из параметров ini-файла: <USER>/<PSW>@<DB_NAME> (PSW подставляется в строку в расшифрованном виде).
  Для детализации лога можно запускать программу с параметром. Значение параметра - уровень детализации лога
    - без параметра: самая маленькая детализация;
    - 1: средняя детализация (только ключевые узлы программы);
    - 2: самая подробная детализация с указанием параметров выделения памяти.
